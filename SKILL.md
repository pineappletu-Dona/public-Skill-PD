# 产品开发流程 Skill

## 概述

本 skill 封装了完整的电商产品开发工作流，基于系统化的选品方法论，覆盖从产品认知到执行计划的全流程。

**触发词**：产品开发、选品、市场调研、竞品分析、Amazon选品、跨境电商选品、BSR分析、市场容量分析、价格区间分析、品牌分析、上架时间分析

---

产品开发流程分为六大阶段：

| 阶段 | 内容 | 对应工作表 |

| 1. 产品思维 | 产品认知 / 品牌认知 / 布局认知 / 供应体系 | 产品思维 |
| 2. 市场分析 | 大盘数据（关键词/站点/品类/容量/趋势/热度） | 目录 → 各分析表 |
| 3. 机会点 | 价格区间 / 品牌分析 / 上架时间 / 评论分析 / 卖家属性 | 价格分析 / 品牌分析 / 上架时间 |

## 第一阶段：产品思维

建立对产品的基础认知框架，回答四个核心问题：

### 1）产品认知
分析维度：
- **种类**：产品有哪些细分类型？
- **功效**：核心功能是什么？解决什么问题？
- **设计**：外观、造型、差异化设计点
- **结构**：组装方式、部件构成
- **规格**：尺寸、重量、参数
- **材质**：原材料、手感、耐用性
- **场景**：使用场景（家用/办公/户外/礼品等）

### 2）品牌认知
识别市场上四类品牌及其策略：
- **主导型品牌**：类目龙头，垄断型玩家
- **攻击型品牌**：激进扩张，低价抢占份额
- **追击型品牌**：跟随龙头，快速迭代
- **游击型品牌**：细分缝隙，小批量多款式

### 3）布局认知
分析四类品牌的 online/offline 布局大体情况，理解其渠道策略。

### 4）供应体系
明确供应链模式：
- **OEM**：原始设备制造（代工）
- **ODM**：原始设计制造（代设计+代工）
- **OBM**：自有品牌制造

## 第二阶段：市场分析

### 1）关键词
- 通过关键词挖掘提取市场需求
- 工具推荐：**卖家精灵（关键词挖掘）**
- 目标：找到用户搜索量大的核心词和长尾词

### 2）站点
- 确认该产品的主要 online 市场是哪个站点（Amazon US/UK/EU/JP 等）
- 不同站点的容量和竞争度差异巨大

### 3）品类
- 该站点中该产品的细分子类目有哪些？
- **如果产品可以分为多个细分子类目，则需要调研多个子类目**
- 不要只盯一个类目，要全面覆盖

### 4）市场容量 ← 详见 `references/market-capacity.md`

通过小分类排名预测销量，判定市场容量等级，决定可开发款式数量。

**容量等级判定逻辑**（以美国站为例）：
- 第1名日销 ≥300 且 第50名日销 ≥12 → **大容量** → 可开发 8-10 款
- 第1名日销 ~100 且 第50名日销 ~10 → **大容量** → 可开发 4-6 款
- 第1名日销 ~80 且 第30名日销 ~18 → **中容量** → 可开发 3-4 款
- 第1名日销 ~20 且 第30名日销 3+ → **小容量** → 可开发 1-2 款
- 第1名日销 ~20 且 第10名日销 8+ → **小容量** → 可开发 1 款

英国站和欧盟站有独立的判定标准，详见 `references/market-capacity.md`。

**容量分析五要素**：
1. 总额（类目总销售额）
2. 月均销量
3. 月均销售额
4. 前三名垄断系数
5. 新品占比 / 份额

### 5）市场趋势
- 提取连续三年的数据
- 推算同比增长率
- 判断市场是否处于增长期 / 成熟期 / 衰退期

### 6）市场热度
- 通过 **Google Trends / Facebook / Twitter / Pinterest** 查看搜索趋势
- 找寻产品的风向标及趋势信号

## 第三阶段：机会点挖掘

### 1）价格区间分析 ← 详见 `references/price-analysis.md`

统计前50名 Listing 的产品价格分布，找出销量最好的价格区间：

| 价格区间 | 选品策略 |
|-----------|-----------|
| 58美金以下 | 份额低，需谨慎 |
| 58~68美金 | 份额较高，可考虑 |
| 68~78美金 | 份额中等，需差异化 |
| 78~88美金 | 主力区间，竞争激烈 |
| 88~108美金 | 中高端，利润空间大 |
| 108美金以上 | 高端，需强差异化 |

**分析维度**：各价格区间的产品数量、市场份额占比、平均占比

### 2）品牌分析 ← 详见 `references/brand-analysis.md`

谁卖得最好？垄断系数如何？新品进入率是多少？

**核心分析维度**：
- BSR 排名分布（每个品牌的前多少名）
- Price 价格带分布（品牌定价策略）
- Style 款式类型（台式/塔式/油汀/壁挂式等）
- Sale 销量数据
- 定位（引流款 / 利基款 / 利润款）

**品牌竞争格局判断**：
- 前三名品牌合计份额 > 60% → 高度垄断，新品进入难
- 前三名品牌合计份额 < 40% → 分散市场，机会较大
- 新品占比 > 20% → 市场活跃，新玩家有机会

### 3）上架时间分析 ← 详见 `references/shelf-time.md`

**老品霸榜率** = 上架3年以上的产品占据多少份额

| 上架时间 | 选品含义 |
|-----------|-----------|
| 3年及以上老品占比高（>60%） | 市场固化，需强差异化切入 |
| 1~2.5年产品有一定占比 | 中坚力量，竞争较激烈 |
| 半年以内新品占比高（>15%） | 市场活跃，新玩家有机会 |

**卖家属性分析**：
- Amazon 自营占比（是否垄断类目）
- 第三方卖家占比 & 新品上架率
- FBA 发货占比 & 新品 FBA 占比

### 4）评论分析
- 市场存在的痛点（低分评论的核心投诉点）
- 市场存在的空白（用户需求未被满足的点）
- 重点分析 3星/4星评论（期望值与实际体验的差距）

### 5）卖家属性
- Amazon 自营 / FBA / FBM 分布
- 上新率（每月新上架数量）
- 判断类目对新玩家是否友好

## 第四阶段：聚焦分析

### 1）BSR 产品架构分析
子类目中的产品再度细分，按 BSR 排名分段分析产品特征。

### 2）架构拆分（消费者偏好）
深挖消费者的偏好维度：

| 偏好维度 | 分析内容 |
|-----------|-----------|
| 功能偏好 | 消费者最看重哪些功能？ |
| 材质偏好 | 哪种材质最受欢迎？ |
| 颜色偏好 | 哪些颜色销量最好？ |
| 尺寸偏好 | 主流尺寸是什么？ |
| 形状偏好 | 哪些造型更受欢迎？ |

→ 消费心理需求及购买行为是产品定位的根基

### 3）竞品分析
- **不同阶段的竞品锁定**：Top 10 / Top 11-30 / Top 31-50 分别分析
- 各阶段竞品的产品特征、定价、卖点、差评

### 4）痛点卖点分析
- 竞品面临的痛点（差评核心问题）
- 竞品的优点（好评核心原因）
- 找到可以做得更好的点

---

## 第五阶段产品定位

### 1）确认开发的款式
- **痛点深挖**：基于第四阶段的分析，列出可解决的痛点
- **如何应对痛点及优化**：具体的产品改进方案
- **优化设计思路**：差异化切入角度
- **如何挖掘差异化**：找到竞品未覆盖的空白点
- **效果图/样品**：构思与完成度确认
- **规格书**：如何生产，如何成本最小化
- **BOM**（Bill of Materials）：找到成本的底层逻辑
- **卖点总结**：找出竞对优势，提炼自己的独特卖点

### 2）利润分析对比
多维度核算毛利：
- 产品成本（BOM成本）
- 头程运费
- FBA 配送费
- 佣金
- 广告费（预估 CPC）
- 其他费用
- **目标毛利率**：通常 > 30%

---

## 第六阶段：执行计划

### 1）销售计划
- 预计首单销量
- 预计月销量（基于市场容量分析）
- 定价策略（基于价格区间分析）
- 推广节奏（广告/秒杀/优惠券）

### 2）备货计划
- 针对市场容量定制开款计划
- 首单数量（基于销售计划）
- 补货周期（基于物流时效）
- 安全库存设定

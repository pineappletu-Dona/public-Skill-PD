# 品类提示词模板库 Skill

## 这是什么

一套**可直接复制给AI执行的提示词模板**，覆盖电商产品开发P1-P6全流程，按**四大品类**分别定制差异化指令。

### 核心价值

| 痛点 | 本Skill的解法 |
|------|--------------|
| 每次让AI分析都要从头描述需求 | 复制对应Phase的提示词 → 替换3个占位符 → 直接发 |
| 不同品类的分析维度容易遗漏 | 每个Phase都有四品类"增强包"，自动覆盖该品类特殊关注点 |
| 分析产出格式不统一 | 每个提示词都定义了输出格式（表格/清单/打分卡） |
| 不知道该采集哪些字段 | 每个阶段附"字段映射表"，对照欧孕SOP的43列逐项检查 |

### 与其他Skill的关系

```
你的使用路径：

① "帮我做[某品类]的市场分析"
   ↓
② WorkBuddy加载 product-dev-workflow（确定方向）
   ↓
③ 需要深入时加载 ouyun-sgrade-market-analysis（走完全流程）
   ↓
④ 具体操作时加载本Skill → 复制提示词 → 给AI执行
```

## 零、通用前置：品类定位五问

每个新品类开发前，**必须先回答5个问题**：

| # | 维度 | 问题 | 决策影响 |
|---|------|------|----------|
| 1 | 类目定位 | 这个品类是拉新还是复购？ | 营销预算方向 |
| 2 | 季节属性 | 季节性如何？全年占比预估 | 开发节奏 & 备货周期 |
| 3 | 消费属性 | 消费频次？单客年度消费额？ | SKU数量 & 价格带宽度 |
| 4 | 客群画像 | 核心客群年龄/月龄/场景？ | 抗风险能力 & 投放方向 |
| 5 | 生命周期 | 产品使用周期？替换频率？ | LTV模型预估 |

## Phase 1: 市场分析

### 1.1 通用提示词

```
你是一位电商数据分析师。请对【{品类名称}】进行完整的市场大盘分析。

## A. 大盘概况（数据源：情报通BI / 生参）
1. 提取近12个月该品类月度销售额趋势，计算各月占比
2. 判断品类处于：成长期 / 成熟期 / 衰退期？
3. 识别峰值月份和谷底月份

### B. 关键词分析（数据源：生参搜索排行）
1. 提取TOP50核心搜索词及搜索人气
2. 分类整理：一级大类词 / 精准长尾词 / 新兴热点词
3. 计算各词CTR和CVR中位数

### C. 输出格式
- Markdown表格呈现
- 最后给出3条关键结论 + 1条风险预警
```

### 1.2 各品类增强指令摘要

| 品类 | 追加的核心分析项 | 关键产出物 |
|------|------------------|-----------|
| **🍼母婴** | 安全合规扫描(CPC/FDA/A类/OEKO) + 月龄段分层(新生儿/婴儿/幼儿) + 决策角色链分析 | 《安全合规检查清单》 + 《月龄段×价格带热力图》 |

## Phase 2: 机会点挖掘

### 2.1 通用提示词

```
基于P1的大盘结论，对【{品类名称}】进行机会点深度挖掘。

## A. 价格带机会分析
将TOP100产品按价格段分组统计：
-每组输出：SKU数量 / 平均销量 / 平均销售额 / 竞争强度
- 找出"高销量+低竞争"的价格带 → 标记⭐机会价格带

## B. 品牌格局分析
- CR5 & CR10 → 判断：寡头型(>60%) / 竞争型(30-60%) / 分散型(<30%)
- 分析头部品牌"防守漏洞"

## C. 评论痛点挖掘（关键！）
采集Top10竞品差评(各≥50条)，词频统计Top20痛点：
- 🔴质量问题(必须避免) / 🟡体验缺陷(=卖点) / 🟢未满足需求(蓝海)

## D. 输出：《机会评分卡》，每方向打分(1-10)
```

## Phase 4: 产品定位（⭐ 含欧孕四级定价公式）

### 4.1 通用提示词

```
基于P1-P3全部结论，完成【{产品名}】的产品定位方案。

## A. SKU规划 — SPU数/SKU数/优先级(主推/形象/利润/引流)
## B. 成本核算明细 — 逐项列式(材料/加工/包装/模具/检测/国内运/头程)
## C. 定价测算（四级模型）

层级:   日常(系数1.0) / A级(0.9) / A+(0.88) / S级(0.85)

公式:
  到手价 = 前台价 × 折扣系数 - 优惠券
  扣点   = 到手价 × 平台费率 (淘系5.5%, 跨境8-15%)
  毛利额 = 到手价 × (1-退款率) - 扣点 - 快递 - 成本 - 到手价 × 营销费比
  毛利率 = 毛利额 / 到手价
  保本ROI = 1 / 毛利率

## D. 利润&ROI三情景测算（保守/中性/乐观）
## E. 差异化卖点最终版（主卖点1句 + 副卖点3-4条 + 信任背书）

产出：SKU矩阵表 + 成本明细Excel + 四级定价表 + ROI测算表 + 卖点卡片
```

### 4.2 各品类特殊处理

| 品类 | P4核心决策 | 成本项目差异 |

| **母婴** | 年龄段切割策略 + 包装礼品化 + 安全信任传达 | +认证费用/+检测报告摊销 |
| **服饰** | 首波SKU≤3色×≤4码=≤12SKU + 退货压降措施 | +面料锁定/+色牢度测试/+次品率约定 |
| **塑料** | 公模测款→私模起量决策 + 品质感低成本提升 | +开模摊销(公模vs私模) |
| **大件** | 物流极限优化 + 价值感塑造 + 安装门槛降到最低 | +包装测试(ISTA6)+/海外仓协同 |

## 附录速查

### A. 核心公式一览

```
【定价类】
到手价   = 前台价 × 折扣系数 - 优惠券
毛利额   = 到手价×(1-退款率) - 扣点 - 快递 - 成本 - 到手价×营销费比
毛利率   = 毛利额 / 到手价
保本ROI   = 1 / 毛利率
定价倍率 = 售价 / 成本

【销售模型】
月销售额  = 年度目标 × 月大盘占比
月需访客  = 月销售额 / 客单价 / 转化率
库存成本 = (月销×1.3/均价/30) × 备货天数

【产品布局】
全渠道月销 = SUM(各渠道)
销售额预估 = 总月销 × 均价
营销费用   = 销售额预估 × 费比率(新品30%→稳定13-15%)
```

### B. 各品类完整阈值表

| 指标 | 母婴 | 服饰 | 塑料 | 大件 |
|------|------|------|------|------|
| 目标毛利率 | ≥30% | ≥35% | ≥20% | ≥28% |
| 保本ROI | ≤3.3 | ≤2.9 | ≤5.0 | ≤3.6 |
| 新品营销费比 | 25-35% | 25-30% | 20-30% | 20-28% |
| 稳定营销费比 | 12-18% | 10-15% | 10-15% | 12-16% |
| 目标退款率 | ≤10% | ≤25% | ≤5% | ≤8% |
| 安全库存天数 | 45天 | 30天(季) | 40天 | 55天 |
| 备货系数 | 1.3 | 1.2 | 1.3 | 1.3 |

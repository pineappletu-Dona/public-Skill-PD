# 欧孕S级市场分析&产品开发 - SOP流程

## 适用范围

- **S级/S+级品类**：必须完整走完全流程
- **非S级品类**：可简化，仅做大盘概况+核心竞品分析
- **品牌**：欧孕（孕产妇用品）
- **平台**：淘系（天猫/淘宝）为主，抖音为辅

---

## 一、SOP总览 — 六大阶段

| 阶段 | 编号 | 板块名称 | 责权人 | 核心产出 |
|------|------|----------|--------|----------|
| **阶段一** | A+ | 品类拓展 SOP大纲 | 品类负责人 | 全年品类规划、开发优先级排序 |
| **阶段二** | A1-A4 | 开品需求（SKU/SPU/市场机会） | 品类负责人 | SKU名称、色系、设计参考、客群、场景 |
| **阶段三** | B1-B6 | 核心分析（竞品购买→SKU布局） | 运营组长 | 竞品定位、销售目标、销售模型、SKU布局逻辑 |
| **阶段四** | C1-C5 | 数据采集（TOP单链/市场提炼） | 运营助理 | TOP100数据、主图、搜索词、单链拆解 |
| **阶段五** | 1-10 | 定价定级→单品拆解 | 运营组长+助理 | 定价表、产品布局、销售模型、竞品卖点、评价提炼 |
| **阶段六** | 一/二 | 淘系/孕产大盘概况 | 运营助理 | 行业趋势、品类容量、月度占比 |

## 二、阶段一：品类拓展 SOP大纲

### 1.1 品类定位维度

在做任何新品类拓展前，必须先回答 **5 个定位问题**：

| 序号 | 维度 | 核心问题 | 决策影响 |
|------|------|----------|----------|
| 1 | 类目定位 | 品类拉新 vs 客群复购？ | 决定营销预算分配方向 |
| 2 | 类目季节 | 产品线季节性如何？品牌客群年度占比预估 | 决定开发节奏&备货周期 |
| 3 | 消费属性 | 消费频次&消费季节性？品类预估 | 决定SKU数量&价格带宽度 |
| 4 | 客群年龄 | 品牌客群月龄段分布？ | 预警：出生率降低&抗风险能力 |
| 5 | 消费频次 | 品类生命周期&产品消费周期？ | 数据化客群画像模型预估 |

### 1.2 岗位责权矩阵

```
【大盘概况】（站内外整体分析）
├── 品类负责人：A1~A4（SKU名称策划、色系开发、设计参考、消费客群）
├── 运营组长：B1~B6（竞品购买、销售目标、销售模型、消费场景/频次/布局逻辑）
└── 运营助理：C1~C5（TOP单链采集、市场提炼、材质价格带、卖点提炼、站外流量）

【品类策略&单品布局】（产品线初稿策划）
├── 品牌部负责 + 品类负责人共同确认
└── S/S+级品类：必须做精细化SKU布局分析&花色稿件&开发布局
```

## 五、阶段四：定价定级 & 销售模型

### 5.1 模块1【定价定级-逻辑|竞对】

#### 定价四大原则

| 原则 | 内容 |
|------|------|
| 原则1 | 整体价格定位中端，参考毛利、竞品、市场环境和自身情况 |
| 原则2 | 定价确认后一般不允许破价；特殊破价需提前申请 |
| 原则3 | A级和S级活动采用立减金额方式，确保日常/A级/S级渠道价拉通 |
| 原则4 | 淘客/挑客以单链接1件为主，2件及以上需另行申请 |

#### 定价表结构（含公式逻辑）

**表头字段**：

| 字段 | 公式/说明 |
|------|-----------|
| 序列 | 1/2/3... |
| 件数 | 1件/3条包装等 |
| 产品名称 | SKU名称 |
| 产品定级 | S/A/B |
| 到手价\|倍率 | `= 前台价 - 优惠券` |
| 款式编码 | OY1-25AW-XXXXX-X |
| 颜色 | 颜色SKU |
| 规格 | 如 80-130 |
| 成本价 | 采购/生产成本 |
| 前台价 | 标价 |
| 优惠券 | 券面额 |
| **日常到手价** | **`= 前台价 - 优惠券`** |
| **日常扣点** | **`= 到手价 × 5.5%`** |
| 快递费 | 固定值（如2.5） |
| **日常毛利额** | **`= 到手价 × (1-退款率) - 扣点 - 快递费 - 成本价 - 到手价 × 营销费比`** |
| **日常毛利率** | **`= 毛利额 / 到手价`** |
| 退款率 | 默认 0.1~0.15 |
| 营销费比 | 默认 0.1 |
| **实际保本ROI** | **`= 1 / 毛利率`** |
| **保本ROI费比** | **`= 毛利率`**（最大值 = 营销费比） |

#### 活动层级定价公式

每个活动层级（A级/A+级/S级）复用相同逻辑，仅替换折扣系数：

| 层级 | 折扣系数 | 到手价公式 | 说明 |
|------|----------|------------|------|
| 日常 | 1.0 | 前台价 - 优惠券 | 原价销售 |
| A级活动 | 0.9 | 前台价 × 0.9 - 优惠券 | 10%立减 |
| A+级活动 | 0.88 | 前台价 × 0.88 - 优惠券 | 12%立减 |
| S级活动 | 0.85 | 前台价 × 0.85 - 优惠券 | 15%立减 |

> **公式模板**（以A级为例）：
> ```
> A级到手价 = 前台价 × 0.9 - 优惠券
> A级扣点  = A级到手价 × 5.5%
> A级毛利额 = A级到手价 × (1-退款率) - 扣点 - 快递费 - 成本 - 到手价 × 营销费比
> A级毛利率 = A级毛利额 / A级到手价
> 保本ROI   = 1 / 毛利率
> ```

#### 竞品对标区

| 字段 | 说明 |
|------|------|
| 对标店铺 | 竞品店铺名 |
| 卖点 | 核心卖点 |
| 规格 | 如睡袋尺码 |
| 日常到手价 | 竞品日常价 |
| A级/S级活动到手价 | 竞品活动价 |

#### 月度拆解公式（按大盘月占比分配）

```
月销售额    = 年度目标销售额 × 对应月份大盘占比
月推广费    = 月均付费访客 × 付费访客占比 × PPC
月需访客    = 月销售额 / 客单价 / 转化率
月均单量    = 月销售额 / 客单价
月均付费访客 = 月需访客 × 付费访客占比
日销售额    = 月销售额 / 30
日推广费    = 月推广费 / 30
日需访客    = 月需访客 / 30
日均单量    = 月均单量 / 30
日均付费访客 = 月均付费访客 / 30
```

#### 产品库存成本公式

```
产品库存成本（130%备货率） = 月销售额 × 1.3 / 30 × 备货可售天数
```

#### 月度经营指标

| 指标 | 公式 |
|------|------|
| 月度营销费比 | `= 月推广费 / 月销售额`（初期0.258 → 稳定期0.134） |
| 月度毛利润 | `= 月销售额 × 毛利率` |
| 月度毛利润占比 | `= 月毛利润 / 月销售额`（初期0.373 → 稳定期0.497） |

#### 大盘占比数据源

```
数据来源：情报通BI
查询路径：行业分析 → 行业数据
查询条件：三级类目 + 年度时间范围
提取字段：每月销售额 + 计算月占比
```

#### 假设费用模型

| 阶段 | 营销费比 | 说明 |
|------|----------|------|
| 链接累计权重期（1-3个月） | 25.8% | 新品起量期，高投入 |
| 链接稳定期（3个月后） | 13.4% | 稳定经营期 |

## 九、核心公式速查

### 9.1 定价类公式

```
到手价 = 前台价 × 折扣系数 - 优惠券
扣点   = 到手价 × 5.5%
毛利额 = 到手价 × (1 - 退款率) - 扣点 - 快递费 - 成本价 - 到手价 × 营销费比
毛利率 = 毛利额 / 到手价
保本ROI = 1 / 毛利率
定价倍率 = 产品定价 / 成本需求
```

### 9.2 销售模型公式

```
月销售额     = 年度目标 × 月大盘占比
月推广费     = 月均付费访客 × 付费访客占比 × PPC
月需访客     = 月销售额 / 客单价 / 转化率
月均单量     = 月销售额 / 客单价
产品库存成本 = 月销售额 × 1.3 / 30 × 备货天数
月度营销费比 = 月推广费 / 月销售额
月度毛利润   = 月销售额 × 毛利率
```

### 9.3 产品布局公式

```
定价倍率          = 价格带平均件单价 / 预估成本
全渠道总月销      = SUM(天猫 + 京东 + 拼多多 + 抖音各渠道)
销售额预估        = 全渠道总月销 × 价格带平均件单价
月度平均库存货值  = (全渠道总月销 × 1.3 × 均价 / 30) × 备货可售天数
营销费用预估      = 销售额预估 × 30%
```

# 产品开发流程 Skill

## 概述

本 skill 封装了完整的电商产品开发工作流，基于系统化的选品方法论，覆盖从产品认知到执行计划的全流程。

**适用平台**：Amazon（主）/ 淘系天猫 / 抖音 / 跨境全渠道
**支持品类**：母婴用品 / 服装服饰 / 塑料制品 / 中大件产品
**关联Skill**：
- `ouyun-sgrade-market-analysis` → 淘系S级品类深度SOP（18表/43列/定价公式）
- `category-prompts` → 四大品类分阶段提示词模板库（可直接复制给AI执行）

**触发词**：产品开发、选品、市场调研、竞品分析、Amazon选品、跨境电商选品、BSR分析、市场容量分析、价格区间分析、品牌分析、上架时间分析

---

## 目录（工作流总览）

产品开发流程分为六大阶段：

| 阶段 | 内容 | 对应工作表 |
|------|------|-------------|
| 1. 产品思维 | 产品认知 / 品牌认知 / 布局认知 / 供应体系 | 产品思维 |
| 2. 市场分析 | 大盘数据（关键词/站点/品类/容量/趋势/热度） | 目录 → 各分析表 |
| 3. 机会点 | 价格区间 / 品牌分析 / 上架时间 / 评论分析 / 卖家属性 | 价格分析 / 品牌分析 / 上架时间 |
| 4. 聚焦分析 | BSR产品架构 / 架构拆分 / 竞品分析 / 痛点卖点 | — |
| 5. 产品定位 | 确认开发款式（痛点深挖/优化设计/效果图/规格书/BOM/卖点） / 利润分析 | — |
| 6. 执行计划 | 销售计划 / 备货计划 | — |

## 第一阶段：产品思维

建立对产品的基础认知框架，回答四个核心问题：

### 1）产品认知
分析维度：
- **种类**：产品有哪些细分类型？
- **功效**：核心功能是什么？解决什么问题？
- **设计**：外观、造型、差异化设计点
- **结构**：组装方式、部件构成
- **规格**：尺寸、重量、参数
- **材质**：原材料、手感、耐用性
- **场景**：使用场景（家用/办公/户外/礼品等）

### 2）品牌认知
识别市场上四类品牌及其策略：
- **主导型品牌**：类目龙头，垄断型玩家
- **攻击型品牌**：激进扩张，低价抢占份额
- **追击型品牌**：跟随龙头，快速迭代
- **游击型品牌**：细分缝隙，小批量多款式

### 3）布局认知
分析四类品牌的 online/offline 布局大体情况，理解其渠道策略。

### 4）供应体系
明确供应链模式：
- **OEM**：原始设备制造（代工）
- **ODM**：原始设计制造（代设计+代工）
- **OBM**：自有品牌制造

## 第二阶段：市场分析

### 1）关键词
- 通过关键词挖掘提取市场需求
- 工具推荐：**卖家精灵（关键词挖掘）**
- 目标：找到用户搜索量大的核心词和长尾词

### 2）站点
- 确认该产品的主要 online 市场是哪个站点（Amazon US/UK/EU/JP 等）
- 不同站点的容量和竞争度差异巨大

### 3）品类
- 该站点中该产品的细分子类目有哪些？
- **如果产品可以分为多个细分子类目，则需要调研多个子类目**
- 不要只盯一个类目，要全面覆盖

### 4）市场容量 ← 详见 `references/market-capacity.md`

通过小分类排名预测销量，判定市场容量等级，决定可开发款式数量。

**容量等级判定逻辑**（以美国站为例）：
- 第1名日销 ≥300 且 第50名日销 ≥12 → **大容量** → 可开发 8-10 款
- 第1名日销 ~100 且 第50名日销 ~10 → **大容量** → 可开发 4-6 款
- 第1名日销 ~80 且 第30名日销 ~18 → **中容量** → 可开发 3-4 款
- 第1名日销 ~20 且 第30名日销 3+ → **小容量** → 可开发 1-2 款
- 第1名日销 ~20 且 第10名日销 8+ → **小容量** → 可开发 1 款

英国站和欧盟站有独立的判定标准，详见 `references/market-capacity.md`。

**容量分析五要素**：
1. 总额（类目总销售额）
2. 月均销量
3. 月均销售额
4. 前三名垄断系数
5. 新品占比 / 份额

### 5）市场趋势
- 提取连续三年的数据
- 推算同比增长率
- 判断市场是否处于增长期 / 成熟期 / 衰退期

### 6）市场热度
- 通过 **Google Trends / Facebook / Twitter / Pinterest** 查看搜索趋势
- 找寻产品的风向标及趋势信号

## 第三阶段：机会点挖掘

### 1）价格区间分析 ← 详见 `references/price-analysis.md`

统计前50名 Listing 的产品价格分布，找出销量最好的价格区间：

| 价格区间 | 选品策略 |
|-----------|-----------|
| 58美金以下 | 份额低，需谨慎 |
| 58~68美金 | 份额较高，可考虑 |
| 68~78美金 | 份额中等，需差异化 |
| 78~88美金 | 主力区间，竞争激烈 |
| 88~108美金 | 中高端，利润空间大 |
| 108美金以上 | 高端，需强差异化 |

**分析维度**：各价格区间的产品数量、市场份额占比、平均占比

### 2）品牌分析 ← 详见 `references/brand-analysis.md`

谁卖得最好？垄断系数如何？新品进入率是多少？

**核心分析维度**：
- BSR 排名分布（每个品牌的前多少名）
- Price 价格带分布（品牌定价策略）
- Style 款式类型（台式/塔式/油汀/壁挂式等）
- Sale 销量数据
- 定位（引流款 / 利基款 / 利润款）

**品牌竞争格局判断**：
- 前三名品牌合计份额 > 60% → 高度垄断，新品进入难
- 前三名品牌合计份额 < 40% → 分散市场，机会较大
- 新品占比 > 20% → 市场活跃，新玩家有机会

### 3）上架时间分析 ← 详见 `references/shelf-time.md`

**老品霸榜率** = 上架3年以上的产品占据多少份额

| 上架时间 | 选品含义 |
|-----------|-----------|
| 3年及以上老品占比高（>60%） | 市场固化，需强差异化切入 |
| 1~2.5年产品有一定占比 | 中坚力量，竞争较激烈 |
| 半年以内新品占比高（>15%） | 市场活跃，新玩家有机会 |

**卖家属性分析**：
- Amazon 自营占比（是否垄断类目）
- 第三方卖家占比 & 新品上架率
- FBA 发货占比 & 新品 FBA 占比

### 4）评论分析
- 市场存在的痛点（低分评论的核心投诉点）
- 市场存在的空白（用户需求未被满足的点）
- 重点分析 3星/4星评论（期望值与实际体验的差距）

### 5）卖家属性
- Amazon 自营 / FBA / FBM 分布
- 上新率（每月新上架数量）
- 判断类目对新玩家是否友好

---

## 第四阶段：聚焦分析

### 1）BSR 产品架构分析
子类目中的产品再度细分，按 BSR 排名分段分析产品特征。

### 2）架构拆分（消费者偏好）
深挖消费者的偏好维度：

| 偏好维度 | 分析内容 |
|-----------|-----------|
| 功能偏好 | 消费者最看重哪些功能？ |
| 材质偏好 | 哪种材质最受欢迎？ |
| 颜色偏好 | 哪些颜色销量最好？ |
| 尺寸偏好 | 主流尺寸是什么？ |
| 形状偏好 | 哪些造型更受欢迎？ |

→ 消费心理需求及购买行为是产品定位的根基

### 3）竞品分析
- **不同阶段的竞品锁定**：Top 10 / Top 11-30 / Top 31-50 分别分析
- 各阶段竞品的产品特征、定价、卖点、差评

### 4）痛点卖点分析
- 竞品面临的痛点（差评核心问题）
- 竞品的优点（好评核心原因）
- 找到可以做得更好的点

---

## 第五阶段产品定位

### 1）确认开发的款式
- **痛点深挖**：基于第四阶段的分析，列出可解决的痛点
- **如何应对痛点及优化**：具体的产品改进方案
- **优化设计思路**：差异化切入角度
- **如何挖掘差异化**：找到竞品未覆盖的空白点
- **效果图/样品**：构思与完成度确认
- **规格书**：如何生产，如何成本最小化
- **BOM**（Bill of Materials）：找到成本的底层逻辑
- **卖点总结**：找出竞对优势，提炼自己的独特卖点

### 2）利润分析对比
多维度核算毛利：
- 产品成本（BOM成本）
- 头程运费
- FBA 配送费
- 佣金
- 广告费（预估 CPC）
- 其他费用
- **目标毛利率**：通常 > 30%

---

## 第六阶段：执行计划

### 1）销售计划
- 预计首单销量
- 预计月销量（基于市场容量分析）
- 定价策略（基于价格区间分析）
- 推广节奏（广告/秒杀/优惠券）

### 2）备货计划
- 针对市场容量定制开款计划
- 首单数量（基于销售计划）
- 补货周期（基于物流时效）
- 安全库存设定

## 第七阶段：四品类差异化分析（⭐ v1.1 新增）

> 不同品类在六个阶段中的关注点差异巨大。以下矩阵指导你在各阶段针对特定品类做**差异化分析**。
>
> **完整提示词模板**请调用 `category-prompts` Skill，以下为快速参考。

### 7.1 品类核心特征速查

| 维度 | 母婴用品 | 服装服饰 | 塑料制品 | 中大件产品 |
|------|----------|----------|----------|------------|
| **核心风险** | 安全合规 | 季节库存 | 同质化价格战 | 物流成本占比 |
| **利润空间** | 高(30-50%) | 中高(35-45%) | 低(15-25%) | 中(25-35%) |
| **SKU策略** | 少而精 | 控制变体数 | 主推1-2款 | 1款打透 |
| **起步资金** | 3-5万 | 5-8万 | 2-3万 | 8-15万 |
| **回本周期** | 3-4月 | 2-3月 | 2-3月 | 4-6月 |
| **最大坑** | 认证不全下架 | 旺季断货 | 品质客诉暴增 | 尾程超费 |
| **复购率** | 高 | 中 | 低 | 极低 |

### 7.2 各阶段品类差异化指令

#### P1 市场分析 — 差异化

| 分析维度 | 母婴 | 服饰 | 塑料 | 大件 |
|----------|------|------|------|------|
| **额外分析项** | 安全认证(CPC/3C/FDA/A类) | 季节性波动模型/提前量 | 材质格局(PP/PE/PET/ABS) | 物流成本敏感性 |
| **关键词特点** | 加安全修饰词(non-toxic/BPA free) | 加季节词+场景词 | 价格敏感度高 | 功能+安装相关词 |
| **趋势判断** | 看法规驱动 | 看时尚周期(提前4-6月) | 看环保升级 | 看住宅/家居趋势 |

#### P2 机会点 — 差异化

| 挖掘方向 | 母婴 | 服饰 | 塑料 | 大件 |
|----------|------|------|------|------|
| **机会来源** | 年龄段空白/安全空白 | 风格空白/尺码空白 | 材质升级/美学化 | 安装优化/空间适配 |
| **差评重点看** | 安全/材质/过敏 | 尺码/色差/面料 | 易碎/异味/易坏 | 运输损坏/安装难 |
| **入局策略** | 认证建立护城河 | 风格差异化切入 | 视觉差异化(低成本高感知) | 服务差异化(质保/安装) |

#### P3 聚焦分析 — 差异化

| 采集重点 | 母婴 | 服饰 | 塑料 | 大件 |
|----------|------|------|------|------|
| **必采字段** | 认证类型/适用月龄/填充物 | 版型/面料克重/尺码覆盖/色系 | 材质/工艺/尺寸精度/承重 | 包装尺寸/净重毛重/安装方式/折叠性 |
| **竞品数量** | 3-5款(深度拆认证) | 5-8款(广度看款式趋势) | 5-10款(看同质化程度) | 3-5款(每款成本高) |
| **透视维度** | 月龄×场景×认证等级 | 风格×尺码×面料×季节 | 材质×价格带×工艺 | 尺寸段×价格带×功能 |

#### P4 产品定位 — 差异化

| 定位要素 | 母婴 | 服饰 | 塑料 | 大件 |
|----------|------|------|------|------|
| **SKU控制** | 按年龄段切分 | ≤3色×≤4码=≤12SKU | 公模测款→私模起量 | 1-2个规格打透 |
| **目标毛利率** | ≥30% | ≥35% | ≥20% | ≥28% |
| **成本特殊项** | 认证费/检测报告摊销 | 面料锁定/次品率约定 | 开模摊销(公模vs私模决策) | 物流极限优化/包装测试 |
| **定价溢价点** | 安全背书/礼品属性 | 设计感/面料功能 | 工业品质平民价 | 质保延长/服务包 |

#### P5 供应链 — 差异化

| 供应链重点 | 母婴 | 服饰 | 塑料 | 大件 |
|-------------|------|------|------|------|
| **供应商核心要求** | 材质安全(COA/SGS报告) | 面料锁定+色牢度+急单能力 | 模具所有权/原料指定/注塑参数 | 跌落测试(ISTA6)/CKD vs SKD |
| **样品验收重点** | 无异味/无小零件/耐拉扯 | 色差/尺寸/手感/耐洗 | 尺寸公差/重量/表面处理 | 包装完整性/配件齐全 |
| **MOQ建议** | 300-500 | 150-300/色 | 500-1000 | 50-100 |
| **谈判重点** | 批次一致性/留底制度 | A等品率≥95%/退换条款 | 模具归属权/备模费用 | 海外仓协同/调拨成本 |

#### P6 执行落地 — 差异化

| 运营重点 | 母婴 | 服饰 | 塑料 | 大件 |
|----------|------|------|------|------|
| **Listing首屏** | 认证标识/检测报告截图 | 尺码表强化/多角度试穿 | 创意摆拍/暴力测试视频 | 尺寸参照图/10年质保/FAQ前置 |
| **广告策略** | DMP人群包(孕妇/宝妈) | 节奏铁律(提前T-10周) | 性价比传达/组合销售 | CPC高→抓转化/再营销重要 |
| **库存预警** | 安全库存45天 | 热门码60天/冷门30天 | 安全库存40天 | 安全库存55天 |
| **节点营销** | 母亲节/百日宴/满月礼 | 换季节奏(严格按周推进) | 无明显季节→持续投放 | 家居节/大促集中爆发 |

### 7.3 品类选择决策树

```
你的产品属于哪个品类？
│
├─ A. 涉及婴幼儿使用 → 🍼 母婴用品
│   └─ 先跑 safety-compliance check，再走P1-P6
│
├─ B. 穿在人身上的 → 👕 服装服饰
│   └─ 先确认季节属性，严格控制SKU数量
│
├─ C. 主要材质是塑料/高分子 → 🧊 塑料制品
│   └─ 先做公模私模决策，再走P1-P6
│
├─ D. 重量>5kg 或 任意边长>50cm → 📦 中大件产品
│   └─ P1阶段优先算物流成本，不通过就换品
│
└─ E. 不确定？→ 用 category-prompts Skill 的"零、通用前置：品类定位五问"
```

### 四、输出格式说明

本 Skill 输出统一遵循以下结构：

1. **阶段标题** → 当前处于哪一阶段
2. **数据表** → 关键数值以 Markdown 表格呈现
3. **公式展示** → 涉及计算时标注 Excel 原始公式来源（来自 `formulas.md`）
4. **判定结论** → ✅ 通过 / ⚠️ 注意 / ❌ 不通过，附带理由
5. **下一步动作** → 明确告诉你该做什么（查什么数据 / 下一步分析什么）

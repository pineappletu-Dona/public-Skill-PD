# 上架时间分析

## 概述

上架时间分析的核心是计算**老品霸榜率**，判断市场是固化还是活跃，从而决定新品牌的进入策略。

---

## 上架时间分段

| 时间段 | 市场含义 |
|---------|-----------|
| 3年及以上 | 老品霸榜，市场固化 |
| 1 ~ 2.5年 | 中坚力量，竞争激烈 |
| 半年以内 | 新品活跃，市场有机会 |

---

## 数据分析（以 Top100 为样本）

### 示例数据（取暖器类目）

| 上架时间 | 产品数 | 市场份额 | 平均占比 |
|-----------|---------|-----------|-----------|
| 3年及以上 | 48个 | 69% | 1.44% |
| 1~2.5年 | 39个 | 17.7% | 0.45% |
| 半年以内 | 13个 | 9.12% | 0.7% |

### 解读

- **老品霸榜率 69%** → 市场高度固化，新品难以撼动
- **新品占比仅 9.12%** → 新玩家进入困难
- **老品平均占比 1.44%** → 老品单品表现强，品牌力强

## 卖家属性分析

### Amazon 自营分析

| 指标 | 含义 |
|------|------|
| 自营产品数 | Amazon 是否重点做这个类目 |
| 自营占比 | 自营垄断程度 |
| 6个月内新品上架数 | 自营是否在这个类目积极扩张 |

> 示例：Amazon 自营 11个产品，占比 3.57%，6个月内无新品上架
> → Amazon 不积极，第三方有机会

### 第三方卖家分析

| 指标 | 含义 |
|------|------|
| 第三方产品数 | 第三方参与程度 |
| 第三方占比 | 非自营空间大小 |
| 新品数 & 占比 | 第三方新玩家活跃度 |

> 示例：第三方 76个产品，新品 13个（占比 9.1%）
> → 第三方有新玩家进入，但比例不高

### FBA 发货分析

| 指标 | 含义 |
|------|------|
| FBA 产品数 | 类目对 FBA 的依赖度 |
| FBA 占比 | FBA 普及率 |
| FBA 新品数 & 占比 | 新玩家用 FBA 的比例 |

> 示例：FBA 33个产品，新品 8个（占比 5.6%）
> → FBA 是主流，但新玩家 FBA 占比不高（可能受资金门槛限制）

## 判断逻辑

### 固化市场（新品难进入）

```
老品霸榜率 > 60%
  AND 新品占比 < 10%
  AND 头部产品 Review 数 > 3000
→ 市场固化，需要强差异化 + 充足推广预算
```

### 活跃市场（新品有机会）

```
老品霸榜率 < 40%
  AND 新品占比 > 20%
  AND 头部产品 Review 数 < 1000
→ 市场活跃，新品有机会
```

### 过渡市场（谨慎进入）

```
老品霸榜率 40%~60%
  AND 新品占比 10%~20%
→ 需仔细分析差异化空间
```

| 工具 | 可获取数据 |
|------|-------------|
| Keepa | 上架时间（Release Date） |
| 卖家精灵 | 上架时间、FBA 标识 |
| Amazon 前台 | 详情页「Date First Available」 |

---

## 输出格式

完成上架时间分析后，输出：

```
【上架时间分析结论】

1. 老品霸榜率：XX%（高/中/低固化）
2. 新品活跃度：XX%（高/中/低活跃）
3. 市场状态：□固化（难进入） □过渡（谨慎） □活跃（有机会）

4. 卖家属性：
   - Amazon 自营占比：XX%（是否垄断？）
   - 第三方新品占比：XX%（新玩家是否成功？）
   - FBA 新品占比：XX%（新玩家资金门槛？）

5. 推荐策略：
   - 若市场固化 → 强差异化 + Vine 快速积累 Review
   - 若市场活跃 → 快速跟进，抢窗口期
   - 若过渡市场 → 仔细分析竞品弱点，精准切入
```

# 品牌分析框架

## 概述

品牌分析的核心：**谁卖得最好？垄断系数如何？新品进入率是多少？**

通过分析头部品牌的定价、款式、销量分布，判断竞争格局和进入机会。

---

## 分析维度

### 1）BSR 排名分布

每个品牌在 Top 50 中有几个产品？排名分布如何？

| 排名段 | 竞争含义 |
|---------|-----------|
| Top 1-10 | 头部霸榜，品牌力强 |
| Top 11-30 | 中坚力量，竞争激烈 |
| Top 31-50 | 有机会切入，差异化空间 |

### 2）Price 价格带分布

每个品牌的定价策略：

| 品牌类型 | 定价特征 |
|-----------|-----------|
| 主导型品牌 | 全价格带覆盖，高中低通吃 |
| 攻击型品牌 | 低价切入，性价比路线 |
| 追击型品牌 | 跟随龙头定价，微幅下调 |
| 游击型品牌 | 聚焦某个窄价格带，做深做透 |

### 3）Style 款式类型

以取暖器为例，常见款式：

| 款式 | 特征 | 适用品牌类型 |
|-------|------|----------------|
| 台式 | 桌面型，小空间 | 游击型 |
| 台式+遥控 | 基础款+便利升级 | 追击型 |
| 塔式+遥控 | 立式，覆盖面积大 | 攻击型 |
| 油汀 | 传统加热，保温好 | 主导型 |
| 壁挂式 | 节省空间，安装型 | 主导型/攻击型 |

→ 统计每个品牌的款式分布，找出**空白款式**

### 4）Sale 销量数据

每个品牌的总销量、平均销量、销量分布：

- 销量是否集中在头部 1-2 个 ASIN？→ 大单品策略
- 销量是否分散在多个 ASIN？→ 矩阵策略
- 新品是否有销量？→ 新品机会

### 5）定位（引流款 / 利基款 / 利润款）

| 定位 | 含义 | 特征 |
|-------|------|------|
| 引流款 | 牺牲毛利，抢流量 | 低价、Review 多、销量高 |
| 利基款 | 细分场景，中等利润 | 特定功能、特定人群 |
| 利润款 | 高毛利，品牌形象 | 高价、高品质、强差异化 |

→ 每个品牌的产品矩阵中，三种定位的配比如何？

---

## 品牌竞争格局判断

### 垄断系数计算

```
垄断系数 = (Top3 品牌销量之和) ÷ (Top50 总销量)
```

| 垄断系数 | 竞争格局 | 新品进入难度 |
|-----------|-----------|----------------|
| > 0.6 | 高度垄断，龙头通吃 | ⭐⭐⭐⭐⭐ 极难 |
| 0.4 ~ 0.6 | 中度集中，三足鼎立 | ⭐⭐⭐ 中等 |
| < 0.4 | 分散竞争，机会较多 | ⭐⭐ 较易 |

### 新品进入率

```
新品进入率 = 上架 < 6个月的产品数 ÷ Top50 总数
```

| 新品进入率 | 市场活跃度 | 含义 |
|-------------|-------------|------|
| > 20% | 非常活跃 | 新玩家不断成功进入 |
| 10% ~ 20% | 中等活跃 | 偶尔有新玩家成功 |
| < 10% | 固化 | 老品霸榜，新玩家难进入 |

## 数据获取

| 工具 | 可获取数据 |
|------|-------------|
| 卖家精灵 | BSR 排名、预估销量、Price、Brand |
| Keepa | 价格历史、上架时间 |
| Amazon 前台 | 详情页、A+ 内容、Review 数 |

---

## 输出格式

完成品牌分析后，输出：

```
【品牌分析结论】

1. 头部品牌（Top3）：
   - 品牌A：XX款产品，BSR 排名 XX~XX，定价 $XX~$XX
   - 垄断系数：XX%（高/中/低）
   
2. 竞争格局：□高度垄断 □中度集中 □分散竞争

3. 空白机会：
   - 价格空白：XX~XX 美金区间无强势品牌
   - 款式空白：XX 款式无头部品牌布局
   - 功能空白：竞品差评中普遍投诉 XX 问题

4. 推荐切入策略：
   - 定位：引流款 / 利基款 / 利润款
   - 定价：$XX（避开头部品牌主力价格带）
   - 差异化：XX（基于空白机会）
```
